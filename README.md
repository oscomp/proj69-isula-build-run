# proj69-isula-build-run

实现在容器中运行isula-build

### *描述*  

目前isula-build 的启动方式是在主机侧启动systemd服务，或手动启动服务进程。理论上isula-build支持在容器内运行，需要进行验证并提供Dockerfile
